# WinRapids

Windows-native GPU-accelerated data science toolkit. Experimental phase.

## Project Philosophy

- **Co-native design**: Representations that work natively for both human and AI agents. No translation layers.
- **Fractal architecture**: Self-similar structures at every scale. The address IS the argument.
- **Backend-as-contract**: A backend is a complete infrastructure package. No fallback. Different backends = different numerical results.
- **Zero tech debt**: See it, fix it. The codebase should always be fully understandable.
- **Reject YAGNI**: We're exploring. Over-engineer the experiments. Find the natural shape.
- **Dual-path design**: Every experiment explores both the pragmatic path (existing libraries) and the from-scratch path (custom CUDA/Rust/C++). Benchmarks decide.
