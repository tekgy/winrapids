## Initial Notes

VB-004 documents the PTX pre-ISA 7.5 composed fallback for min/max NaN. The RTX 6000 Pro Blackwell (SM 120) uses PTX ISA 8.x so Path A (min.NaN.f64) applies. But the pre-7.5 composed sequence is documented for target completeness. Before Peak 3 opens, confirm the campsite pre-flight includes a PTX ISA version check and routes to Path A vs Path B correctly. Also check whether any other PTX ops have NaN-related modifiers analogous to .NaN that we should be using (setp variants, etc).
