## Initial Notes

The three-section tam_exp.toml format implies a taxonomy: bit_exact=wrong sign, constraint=wrong class, injection=wrong propagation. When 3+ oracle TOMLs exist, run a convergence check — do they all need the same three sections, or does a fourth emerge? The schema may want a version field before it proliferates.
