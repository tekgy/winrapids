## Initial Notes

I9 v4 minimum suite includes monotonicity sweep. Current oracle runner has no monotonicity check — only ULP pointwise, bit-exact, constraint, identity. For exp: exp is strictly monotone increasing. A sweep over sorted inputs verifying candidate(x[i]) <= candidate(x[i+1]) would catch non-monotone bugs that pointwise ULP misses.
