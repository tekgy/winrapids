## Initial Notes

Scientist's 2026-04-12 correction (coverage vs policy) gave the condition for when to promote convention → declaration: when convention has grown fragile. But some conventions should be preemptively promoted BEFORE they fail, because the failure mode is silent-wrong-answer rather than loud-bug. Safety-critical conventions where a missed case ships undetected. How do you tell which conventions those are? Incomplete half of the convention-to-declaration finding. Needs investigation. Candidate approach: look at each existing convention in the trek (NaN handling per function, subnormal treatment, rounding mode per op) and classify by failure mode (loud vs silent). Silent-failure conventions get promoted preemptively; loud-failure conventions can wait for the real failure. See deferred-candidates.md and garden entry 2026-04-12-convention-to-declaration.
