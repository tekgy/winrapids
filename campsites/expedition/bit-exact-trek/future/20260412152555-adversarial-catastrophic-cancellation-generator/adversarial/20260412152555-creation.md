Created: 2026-04-12T15:25:55-05:00
By: adversarial

## Initial Notes

Build a generic catastrophic-cancellation input generator for the oracle corpus: given f(x) = a - b where a ≈ b, find x where the true result has magnitude << ulp(a). These are the hardest inputs for any libm function. For exp: x near 0 where exp(x)-1 cancels. For log: x near 1 where log(x) cancels. For sin/cos: x near k*pi/2. The generator should output a TOML injection set usable in any oracle TOML file.
