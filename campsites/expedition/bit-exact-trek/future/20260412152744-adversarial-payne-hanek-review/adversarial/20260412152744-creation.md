## Initial Notes

When Phase 2 implements Payne-Hanek, the adversarial review is nontrivial: (1) the algorithm involves extended-precision arithmetic for argument reduction, which must be I3-compliant (no FMA contraction), (2) the published Payne-Hanek tables have been verified for some ranges but not all, (3) the boundary between Cody-Waite and Payne-Hanek must be smooth to 1 ULP.
