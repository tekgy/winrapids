## Initial Notes

Reading list: Peter Tang's 1989 paper 'Table-driven implementation of the exponential function in IEEE floating-point arithmetic' (ACM TOMS 15:2). Foundational paper on how to implement exp using precomputed tables for the reduced argument, rather than a single polynomial over the full range. This is the approach CRlibm and many modern libms use. Directly relevant to math-researcher's Phase 2+ work if they ever want to squeeze below 1 ULP. Also: Tang's 1992 follow-up for log and Tang's later multi-part series for sin/cos/tan. The whole series is a canonical reference set.
