Each of these is a capability-matrix row that doesn't exist yet. Filing separately because the Vulkan row only covers NVIDIA. When Peak 7 adds a second Vulkan device or Peak 8 (hypothetical) adds Metal, these questions become load-bearing. For now: an observation to carry across sessions.
