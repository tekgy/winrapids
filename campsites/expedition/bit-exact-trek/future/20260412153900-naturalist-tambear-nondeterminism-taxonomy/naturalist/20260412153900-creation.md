## Initial Notes

Taxonomy observation: tambear's five ACTIVE sources of potential cross-backend divergence are: (1) FMA contraction (I3), (2) op reordering for fp (I4), (3) reduction order (I5, with RFA as the solution), (4) NaN propagation (I11), (5) subnormal handling (ESC-001 scope qualifier). Missing from the list that MIGHT matter later: (a) signed zero (campsite), (b) NaN payload / which-NaN (campsite), (c) overflow/underflow flag state (campsite), (d) rounding-mode changes mid-computation (currently impossible in tambear — the IR pins rounding per op — but the attack surface exists if user code ever controls rounding mode). The current five-item list is LIKELY complete for Phase 1 but the full taxonomy of potential divergence sources is worth naming explicitly so nothing gets missed. Draft as a named doc: 'tambear-nondeterminism-taxonomy.md' or similar.
