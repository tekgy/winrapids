## Initial Notes

pi_over_4_hi is NOT piover2_hi/2. Different trailing-zero structures: piover2_hi has 30 trailing zeros (for sin/cos range reduction at |x|<2^30), pi_over_4_hi has 12 (for atan's tighter interval). Halving doesn't preserve the split quality. This is a potential implementation bug if pathmaker writes 'pi_over_4_hi = piover2_hi >> 1' as a shortcut.
