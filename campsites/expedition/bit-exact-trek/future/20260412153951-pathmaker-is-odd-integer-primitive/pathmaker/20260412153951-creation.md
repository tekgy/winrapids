# pathmaker-is-odd-integer-primitive

Created: 2026-04-12T15:39:51-05:00

## Initial Notes

is_odd_integer(b: f64) -> bool appears in pow (negative-base integer branch) AND in the special-values table for pow(-1, ±inf). The fp64 implementation: |b|≤2^53 → floor(b)==b && int(floor(b))%2!=0; |b|>2^53 → false (all large fp64s are even integers). This is a tambear primitive — it should be in the number theory catalog, not a private helper inside pow.
