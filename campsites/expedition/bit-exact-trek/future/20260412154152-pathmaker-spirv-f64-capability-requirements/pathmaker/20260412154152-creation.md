# pathmaker-spirv-f64-capability-requirements

Created: 2026-04-12T15:41:52-05:00
By: pathmaker

## Initial Notes

SPIR-V for Peak 7 needs the Float64 capability declared. But also: every Cody-Waite constant we use must survive the SPIR-V specialization constant path without reinterpretation. The bitcast operations BitcastF64ToI64 / BitcastI64ToF64 map to OpBitcast in SPIR-V, but OpBitcast between integer and float types requires the types to be the same bit-width (both 64-bit). Verify this is spec-compliant before Peak 7.
