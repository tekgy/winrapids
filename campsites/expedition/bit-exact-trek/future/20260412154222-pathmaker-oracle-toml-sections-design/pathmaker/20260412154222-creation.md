## Initial Notes

oracle_runner.rs now has three sections: bit_exact_checks, constraint_checks, identity_checks. A fourth is missing: ulp_checks, where the test passes if |result - reference| <= N ulps. Currently the only way to express 'within 1 ULP' is to generate a specific bit range in bit_exact_checks, which requires knowing the exact result. ULP checks would let us write the oracle without the reference value, just the bound.
