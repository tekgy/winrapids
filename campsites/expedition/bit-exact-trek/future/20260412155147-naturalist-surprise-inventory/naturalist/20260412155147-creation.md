## Initial Notes

Pattern I've been noticing throughout this session: I keep being surprised by things that were already in the repo. The two-naturalist dynamic. Team-lead's convergence-check methodology. The practice file's exemplar finding. Aristotle's synthesis.md. The guarantee-ledger draft. The deferred-candidates document. Every time I looked for something I assumed wasn't there, it was there. Methodology lesson: in a rich session, check BEFORE assuming you're the first to notice. The default assumption should be 'someone else probably already wrote this down' rather than 'I need to write this down first.' The check is cheap; the duplication is expensive.
