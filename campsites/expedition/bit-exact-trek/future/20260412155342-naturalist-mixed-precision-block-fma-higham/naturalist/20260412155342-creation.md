## Initial Notes

Reading: Higham's 'Mixed Precision Block Fused Multiply-Add: Error Analysis and Application to GPU Tensor Cores' (SIAM J. Sci. Comput. 2020). Nick Higham is the standard reference for numerical accuracy of matrix algorithms. This paper specifically addresses the block-FMA pattern used by tensor cores and analyzes the error. Relevant if tambear ever wants to run accumulate operations on tensor cores. Also relevant as a reference for the P2 precondition analysis: 'what does faithful-lowering mean when the target has mixed-precision building blocks?' Filed for Phase 2+. URL: epubs.siam.org/doi/10.1137/19M1289546 (paywalled, also on HAL).
